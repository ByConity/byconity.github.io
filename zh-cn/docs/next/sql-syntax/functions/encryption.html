decryptâ€‹

This function decrypts ciphertext into a plaintext using these modes:

  • aes-128-ecb, aes-192-ecb, aes-256-ecb

  • aes-128-cbc, aes-192-cbc, aes-256-cbc

  • aes-128-cfb1, aes-192-cfb1, aes-256-cfb1

  • aes-128-cfb8, aes-192-cfb8, aes-256-cfb8

  • aes-128-cfb128, aes-192-cfb128, aes-256-cfb128

  • aes-128-ofb, aes-192-ofb, aes-256-ofb

  • aes-128-gcm, aes-192-gcm, aes-256-gcm

Syntax


decrypt('mode', 'ciphertext', 'key' [, iv, aad])

Arguments

  • mode â€” Decryption mode. String .

  • ciphertext â€” Encrypted text that needs to be decrypted. String .

  • key â€” Decryption key. String .

  • iv â€” Initialization vector. Required for -gcm modes, optinal for others. String .

  • aad â€” Additional authenticated data. Won't decrypt if this value is incorrect. Works only in -gcm modes, for others would throw an exception. String .

Returned value

  • Decrypted String. String .
